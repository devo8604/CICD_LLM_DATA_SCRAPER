# Prompt Theme Template

Copy this directory to create a new prompt theme.

## Required Files

- `question_system.txt`: High-level goals for question generation.
- `question_user.txt`: Specific instructions for the model when analyzing content for questions.
- `answer_system.txt`: High-level goals for answer generation.
- `answer_user.txt`: Specific instructions for formatting and answering.

## Customization Tips

- Keep the focus on the content provided.
- Use clear, imperative language.
- Define formatting constraints if the model deviates from expectations.
